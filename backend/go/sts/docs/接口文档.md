##### 1.5.1.1 文件列表

URL地址：

```url
/file/cloud/list
```

请求：

```json
{
    "dirname":"/",
    "file_type":"*"
}
```

响应：

成功：

```json
{
    "state":2000,
    "msg":"success",
    "data":[
        {
            "filepath":"/system/logo.png",
            "is_file":true,
            "file_type":"image/png"
        },
        {
            "filepath":"/clinical/",
            "is_file":false,
            "file_type":"",
        },
        {
            "filepath":"/reports/",
            "is_file":false,
            "file_type":"",
        }
    ]
}
```
